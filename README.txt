## 🚀 Features

| Feature | Description |
|--------|-------------|
| 🔒 **AES-256 Encryption** | Files encrypted before storage using military-grade AES in EAX mode |
| 🗝️ **PBKDF2 Key Derivation** | Secure password-based key generation with salt & 500k iterations |
| 🛡️ **SHA-256 Integrity Check** | Detect tampering by comparing file hash on decryption |
| 📁 **Private Uploads** | Encrypted files stored in secure `uploads/` directory — not publicly accessible |
| 📝 **Intrusion Logging** | All attempts logged with IP, username, action, and status |
| 🌐 **Web-Based UI** | Clean, responsive interface for easy use on any device |
| 🧯 **Auto-Cleanup Ready** | Designed for future auto-delete & file expiry features |
