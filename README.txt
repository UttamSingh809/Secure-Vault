🔐 SECUREVAULT – ENCRYPTED FILE STORAGE

✅ How to Run:
1. Install Python 3.7+
2. Open terminal in this folder
3. Run: pip install flask pycryptodome
4. Run: python app.py
5. Open: http://127.0.0.1:5000

📁 Folders:
- uploads/ → encrypted files
- logs/ → intrusion_log.txt
- templates/ → HTML
- static/ → CSS

🔐 Security:
- Files encrypted with AES-256 via PBKDF2 key derivation
- MD5 used for integrity
- Logs track access attempts
